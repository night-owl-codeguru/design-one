# TJ's Brew Works - Homepage

A brutalist neomorphism design for TJ's Brew Works craft brewery website.

## Features

- **Brutalist Design System**: Bold typography, stark contrasts, and geometric shapes
- **Neomorphic Elements**: Soft shadows and dimensional cards for modern depth
- **Brand Colors**: Golden Pour (#F4A825), Burnt Copper (#C86928), Soft Wheat (#F5E6D3), Dark Roast (#2D1B12)
- **Responsive Layout**: Mobile-first design with smooth animations
- **Component-Based Architecture**: Modular sections for easy maintenance

## Sections

1. **Hero Section**: Eye-catching introduction with "TJ'S TO JOY" headline
2. **About Section**: Brand story and philosophy
3. **Beer Showcase**: Feature 9 craft beer variants
4. **TJ's Tappin'**: Self-pour beer experience
5. **Core Values**: Brand principles and authenticity
6. **Community**: Events, social connections, and experiences
7. **Footer**: Contact information and navigation

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)

## Getting Started

\`\`\`bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build

# Start production server
bun start
\`\`\`

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design Philosophy

**Brutalism meets Neomorphism**: Combines harsh, bold brutalist elements (thick borders, stark shadows, bold typography) with soft neomorphic depth (subtle gradients, layered shadows, dimensional cards) to create a unique "brutal morph" aesthetic that's both edgy and approachableâ€”perfect for TJ's brand identity.

## Brand Guidelines

- **Primary Font**: DIN Next LT Pro (headlines, body)
- **Secondary Font**: Optima Bold (brand line, beer variants)
- **Typography**: Generous kerning, bold weights, uppercase for emphasis
- **Color Usage**: Golden Pour for primary actions, Burnt Copper for accents, Dark Roast for base
- **Visual Style**: "Energy over perfection", "emotion is the hero"
